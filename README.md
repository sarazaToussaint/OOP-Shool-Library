# Oop SchoolL Library

> In this project, I have used Ruby Object Oriented Programming to create a simple School Library
## Getting Started

 - Open your terminal, move to the directory where you want to clone the app (i.e cd Documents)
 - Enter git clone https://github.com/sarazaToussaint/OOP-Shool-Library.git
 - Change to the directory you just clone into (i.e cd OOP-Shool-Library)
 - Include any possible changes and run "git add ."
 - Commit the changes with a descriptive message (i.e xyz features added)
 - Push the changes to changes to the repository (i.e git push)
